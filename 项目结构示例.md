# 📁 冰箱食物管理系统项目结构示例

## 🐳 Docker配置

### 1. 📄 docker/Dockerfile

```dockerfile
# 多阶段构建
FROM mcr.microsoft.com/dotnet/sdk:8.0 AS build
WORKDIR /src

# 复制项目文件
COPY ["src/FridgeMate.API/FridgeMate.API.csproj", "src/FridgeMate.API/"]
COPY ["src/FridgeMate.Core/FridgeMate.Core.csproj", "src/FridgeMate.Core/"]
COPY ["src/FridgeMate.Infrastructure/FridgeMate.Infrastructure.csproj", "src/FridgeMate.Infrastructure/"]
COPY ["src/FridgeMate.Domain/FridgeMate.Domain.csproj", "src/FridgeMate.Domain/"]
COPY ["src/FridgeMate.Shared/FridgeMate.Shared.csproj", "src/FridgeMate.Shared/"]

# 还原包
RUN dotnet restore "src/FridgeMate.API/FridgeMate.API.csproj"

# 复制源代码
COPY . .

# 构建应用
RUN dotnet build "src/FridgeMate.API/FridgeMate.API.csproj" -c Release -o /app/build

# 发布应用
FROM build AS publish
RUN dotnet publish "src/FridgeMate.API/FridgeMate.API.csproj" -c Release -o /app/publish

# 运行时镜像
FROM mcr.microsoft.com/dotnet/aspnet:8.0 AS final
WORKDIR /app
COPY --from=publish /app/publish .

EXPOSE 80
EXPOSE 443

ENTRYPOINT ["dotnet", "FridgeMate.API.dll"]
```

## 📋 关键文件示例

### 1. 📄 Program.cs (应用程序入口)

```csharp
using FridgeMate.API.Extensions;
using FridgeMate.Infrastructure.Data;
using Microsoft.EntityFrameworkCore;

var builder = WebApplication.CreateBuilder(args);

// 添加服务到容器
builder.Services.AddControllers();
builder.Services.AddEndpointsApiExplorer();
builder.Services.AddSwaggerGen();

// 配置数据库
builder.Services.AddDbContext<FridgeMateDbContext>(options =>
    options.UseNpgsql(builder.Configuration.GetConnectionString("DefaultConnection")));

// 配置Redis
builder.Services.AddStackExchangeRedisCache(options =>
{
    options.Configuration = builder.Configuration.GetConnectionString("Redis");
});

// 注册服务
builder.Services.AddApplicationServices();
builder.Services.AddInfrastructureServices();

// 配置CORS
builder.Services.AddCors(options =>
{
    options.AddPolicy("AllowAll", policy =>
    {
        policy.AllowAnyOrigin()
              .AllowAnyMethod()
              .AllowAnyHeader();
    });
});

var app = builder.Build();

// 配置HTTP请求管道
if (app.Environment.IsDevelopment())
{
    app.UseSwagger();
    app.UseSwaggerUI();
}

app.UseHttpsRedirection();
app.UseCors("AllowAll");
app.UseAuthorization();
app.MapControllers();

// 确保数据库已创建
using (var scope = app.Services.CreateScope())
{
    var context = scope.ServiceProvider.GetRequiredService<FridgeMateDbContext>();
    context.Database.Migrate();
}

app.Run();
```

## 📝 开发规范

### 1. 命名规范
- **类名**：PascalCase (如 `FoodItemService`)
- **方法名**：PascalCase (如 `GetFoodItems`)
- **变量名**：camelCase (如 `foodItem`)
- **常量名**：PascalCase (如 `MaxQuantity`)
- **接口名**：I + PascalCase (如 `IFoodItemService`)

### 2. 文件组织
- 每个类一个文件
- 文件名与类名一致
- 按功能模块组织文件夹

### 3. 代码规范
- 使用C#编码规范
- 所有公共方法必须有XML文档注释
- 使用async/await进行异步操作
- 使用依赖注入管理服务

### 4. 测试规范
- 单元测试覆盖率 ≥ 80%
- 使用AAA模式 (Arrange, Act, Assert)
- 测试方法名清晰描述测试场景
